# JavaScript School Library Automation

This is a basic library automation system implemented with Node.js and Express. It provides REST endpoints for managing books, students, and loans.

## Features
- Add, update, delete, and list books
- Search books by title or author
- List available books only
- Register and update students
- Borrow and return books (14-day default due date)
- List active loans
- Track overdue loans

## Usage
Install dependencies and start the server:
```bash
npm install
npm start
```
The server runs on `http://localhost:3000`.
